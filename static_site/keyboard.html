What it does

Wasurenai Keyboard is an alternative keyboard designed to stay familiar to use while collecting the signals needed for gentle progression tracking.

Why it matters

Typing changes can appear before more obvious symptoms. A keyboard can observe those changes early, without adding friction to daily life.

Who it helps

  • People who want a discreet early-warning tool
  • Doctors who need more context sooner
  • Caregivers who want a calmer, clearer picture
